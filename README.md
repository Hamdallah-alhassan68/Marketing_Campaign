📊 Marketing Campaign Performance Dashboard

An interactive Power BI dashboard that analyzes marketing campaign performance, customer behavior, and conversion trends to support data-driven marketing decisions.

📌 Overview

This project evaluates the effectiveness of marketing campaigns by analyzing customer responses, engagement, and conversion rates. It transforms raw marketing data into actionable insights that help optimize campaign strategies.

The dataset used is the Marketing Campaign Dataset.

Objectives
Measure campaign performance and conversion rates
Analyze customer demographics and behavior
Identify high-performing marketing channels
Optimize decision-making through automated reporting
Features
Conversion Rate & Campaign KPIs
Customer Segmentation Analysis (age, income, behavior)
Campaign Performance Comparison
Trend Analysis for campaign effectiveness over time
Interactive Filters & Slicers for dynamic exploration
Tools & Technologies
Data Source: Excel (.csv)
Data Cleaning & Preparation: Microsoft Excel
Data Visualization: Microsoft Power BI
Dataset: Marketing Campaign Dataset
Data Processing Workflow
Data Collection
Imported Marketing_Campaign.csv dataset from Kaggle
Data Cleaning (Excel)
Removed duplicates and handled missing values
Standardized formats (dates, income, categorical variables)
Created calculated fields (e.g., total spend, campaign response)
Data Modeling (Power BI)
Built KPIs such as conversion rate, total customers, and campaign success rate
Created calculated measures for deeper analysis
Dashboard Development
Designed interactive visuals and reports
Implemented slicers for campaign, demographics, and time filtering
Optimized layout for clarity and usability
Key Insights
Identified customer segments with the highest campaign response rates
Revealed which campaigns generated the most conversions
Highlighted patterns between income levels and purchasing behavior
Detected underperforming campaigns needing optimization
Impact
Reduced reporting time by ~30% through automation
Enabled faster marketing decisions using real-time insights
Improved visibility into campaign effectiveness and ROI
